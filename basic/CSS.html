CSS is a styling language(ui) for HTMl.

There are 3 type of CSS

Inline CSS: directly adding CSS to the HTML element.
Ex: (h style="........."),,,,,,,,(/h)

Internal CSS: adding CSS using "class" or "id" between "head" and "body".
Ex:
(/head)
(style)
"Class/Id" {
property: value;
property1: value;
................
}
(body)

Enxternal CSS: linking CSS outside HTML to HTMl file using (link rel="stylesheet" href=",,,,,,.css").
Ex
"Element/Class/Id" {
property: value;
property1: value;
................
}
